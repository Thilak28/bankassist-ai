# BankAssist AI 💳

A full-stack web application that helps users understand their bank transactions and provides official information on how to access mini statements from different banks.

---

## 📌 Problem Statement

Many bank users notice money being debited from their accounts but are unsure why. Transaction descriptions often contain abbreviations that are difficult to understand. In addition, every bank provides different methods to obtain mini statements, making it time-consuming to find the correct official information.

BankAssist AI aims to solve this problem by providing a single platform where users can:

- Find official mini statement methods for multiple banks.
- Upload their own bank statements (PDF/CSV).
- Analyze debit and credit transactions.
- Get explanations for common debit entries.
- View spending insights and recurring charges.
